# 推荐系统引擎 — 目录结构说明

练手用 C++ 项目骨架，**使用 Bazel 3.5 编译**（apt 的 `bazel-bootstrap`）。

```
recommendation_engine/
├── WORKSPACE           # 工作区定义（Bazel 3.x 必需）
├── .bazelrc            # 默认编译选项（如 C++17）
├── .bazelversion       # 锁定 Bazel 版本（3.5.1）
├── BUILD.bazel         # 根 package（通常为空）
├── configs/            # 运行时配置文件（如 ranking.conf）
├── docs/               # 文档（含 Config设计.md）
├── examples/           # 小示例（cc_binary）
├── include/recommendation/  # 对外头文件（cc_library）
├── scripts/            # 辅助脚本
├── src/
│   ├── agent/          # 智能体
│   ├── common/         # 公共类型
│   ├── servers/        # 三个服务器
│   ├── algorithms/     # 推荐算法（cc_library）
│   ├── core/           # 引擎核心（cc_library）
│   ├── data/           # 数据处理（cc_library）
│   └── utils/          # 工具（cc_library）
├── tests/              # 单元测试（cc_test）
├── third_party/        # 第三方源码
└── tools/              # 命令行工具（cc_binary）
```

## Bazel 里几个关键文件

| 文件 | 作用 |
|------|------|
| `WORKSPACE` | 声明这是一个 Bazel 工作区（3.x 必需） |
| `BUILD.bazel` | **每个目录一个**，定义该目录下的编译目标 |
| `.bazelrc` | 全局默认 flags |
| `.bazelversion` | 团队统一 Bazel 版本 |

## 常用命令

```bash
# 编译并运行 demo
bazel build //examples/browse_demo:browse_demo
bazel run //examples/browse_demo:browse_demo

# 编译某个库
bazel build //src/agent:agent

# 编译全部
bazel build //...

# 跑测试（有 cc_test 之后）
bazel test //tests/...
```

## 当前可编译的 target

| Target | 说明 |
|--------|------|
| `//examples/browse_demo:browse_demo` | 完整刷视频流程 demo |
| `//src/agent:agent` | 智能体 |
| `//src/servers/main:main_server` | 主服务器 |
| `//src/servers/ranking:ranking_server` | 排序服务器 |
| `//src/servers/video:video_server` | 拉视频服务器 |
| `//src/common:common` | 公共类型 |
